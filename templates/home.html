Step to add new tool

  1. Create new blueprint (submodule of website) in blueprints folder.
  2. Create __init__.py to set up new blueprint.
  3. Register blueprint in app.py and add the link to that blueprint in global template/layout.html
  4. Create routes.py to control everything about this submodule such as route url to web page (html template) or do some action.
  5. Create templates/page_name.html to create web page (html template is html with capability to inject code into it).
  6. Create static folder to keep static content like css files, images, uploads files etc.
  7. Create logic.py to keep python business logic (python script) for use in routes.py.

Step required after install new packages

  1. install new package by pip install package_name
  2. check all packages installed by pip list
  3. after that run pip freeze > requirements.txt to update all the packages name in the generate requirement.txt file.
